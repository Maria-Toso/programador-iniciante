# Jogo da Forca em Python

Este é um projeto simples de Jogo da Forca desenvolvido para praticar lógica de programação em Python.

### Funcionalidades:
- Escolha aleatória de palavras.
- Interface em texto com arte ASCII.
- Sistema de verificação de letras repetidas.

### Como executar:
Certifica-te de que tens o Python instalado e corre o comando:
```bash
python forca.py
