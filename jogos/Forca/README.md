## Versão 3.0: Interface Gráfica (GUI)
Agora o jogo possui uma interface visual completa utilizando a biblioteca **Tkinter**.

### Melhorias:
- **Interface Visual:** Saída do terminal para uma janela interativa.
- **Gráficos Dinâmicos:** O boneco da forca é desenhado em tempo real num Canvas.
- **Feedback Visual:** Uso de cores (verde para acerto, vermelho para erro) e caixas de diálogo (`messagebox`) para o resultado final.
