# Jogo da Velha (Tic-Tac-Toe) com GUI

Um projeto clássico para exercitar lógica de matrizes e interfaces gráficas em Python.

### Conceitos Aplicados:
- **Programação Orientada a Objetos (POO):** Organização do código em classes.
- **Tkinter Grid:** Posicionamento de elementos em linhas e colunas.
- **Lógica Matemática:** Verificação de padrões de vitória em listas.

### Como jogar:
Basta correr o script e clicar nos quadrados. O jogo alterna automaticamente entre X e O e deteta o vencedor.
